# Getting Started

> This guide walks you through installing ONXZA and creating your first agent.

## Prerequisites

- Node.js 18+ 
- npm or yarn
- A terminal

## Installation

```bash
npm install -g onxza
```

## Initialize a Workspace

```bash
onxza init my-workspace
cd my-workspace
```

This creates a new ONXZA workspace with:
- `openclaw.json` — Configuration file
- `agents/` — Agent definitions
- `companies/` — Company registry
- `shared-learnings/` — Knowledge base
- `tickets/` — Inter-agent task system

## Create Your First Agent

```bash
onxza agent create my-first-agent --company my-company
```
